部署 Apps Script 后得到的 Web App URL。不需要 OAuth 和 Cloud 项目。
如何部署 Apps Script？（点击展开）
  1. 打开 script.google.com（用你的 Google 账号登录）
  2. 点左上角「+ 新建项目」
  3. 删除编辑器中的默认代码，粘贴下方提供的代码
  4. 点「部署」→「新建部署」→ 类型选择「Web 应用」
  5. 执行身份选择「我」，访问权限选「任何人」
  6. 点「部署」，首次会要求授权 → 同意即可
  7. 复制生成的 Web App URL，粘贴到上方输入框
