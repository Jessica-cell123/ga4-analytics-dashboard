# GA4 数据分析工具 — 使用指南

## 📁 工具位置

桌面上的 `GA4-分析工具` 文件夹包含以下文件：
- **dashboard.html** — 数据分析仪表盘（核心工具）
- **使用指南.md** — 本指南

---

## 🚀 快速开始

### 第 1 步：打开工具

**双击**打开 `dashboard.html`，会在浏览器中打开设置向导。

### 第 2 步：找到你的 GA4 属性 ID

1. 打开 https://analytics.google.com
2. 左下角点击 **管理（齿轮图标）**
3. 在「属性」列，第一行就是 **属性 ID**（纯数字）
4. 复制这个数字，粘贴到工具的输入框

### 第 3 步：创建 Google Cloud 项目

1. 打开 https://console.cloud.google.com/projectcreate
2. 项目名称填 `GA4 数据分析`（或其他任意名字）
3. 点击「创建」

### 第 4 步：开启 API 并创建凭据

**开启两个 API（每个点击"启用"按钮即可）：**
- https://console.cloud.google.com/apis/library/analyticsdata.googleapis.com
- https://console.cloud.google.com/apis/library/searchconsole.googleapis.com

**创建 OAuth 凭据：**
1. 打开 https://console.cloud.google.com/apis/credentials
2. 点击「**创建凭据**」→「**OAuth 客户端 ID**」
3. 应用类型选「**Web 应用**」
4. 名称随便填
5. 「已授权的 JavaScript 来源」填你的地址（工具里会自动显示，一般是 `http://localhost` 或 `file://`）

   > ⚠️ **重要**：如果你是用 `file://` 协议打开 HTML，OAuth 可能不支持。建议用 Live Server 或上传到任意静态托管。
   >
   > **最简单的替代方案**：我后面会说明如何用 Python 临时启动一个服务器。

6. 点击「创建」，复制弹出的 **Client ID**

### 第 5 步：连接 Google 账号

1. 把 Client ID 粘贴到工具中
2. 输入你的 Search Console 网站地址
3. 点击「使用 Google 账号登录」
4. 授权后自动跳转回工具

### 第 6 步：开始分析

点击「进入仪表盘」，数据自动加载！

## ⚠️ 常见问题

### Q: 文件打开后无法登录 Google？

因为 OAuth 需要 HTTP 服务，不能直接 `file://` 打开。

**解决方法**（三选一）：

**方法 A：用 Python 启动服务（推荐，无需安装）**
```bash
# 打开终端（按 cmd+空格，输入"终端"回车）
cd ~/Desktop/GA4-分析工具
python3 -m http.server 8080
```
然后在浏览器打开 `http://localhost:8080/dashboard.html`

**方法 B：用 VS Code Live Server**
如果安装了 VS Code，右键 `dashboard.html` → 选择「Open with Live Server」

**方法 C：上传到任意静态托管**
上传到 GitHub Pages、Vercel、Netlify 等都可以

### Q: OAuth 授权页面显示"未验证"？

正常现象，因为这是你自己创建的项目。点击「继续」即可。

### Q: 数据加载不出来？

- 确认已在 GA4 中配置了数据收集
- 确认 API 已启用（第 4 步）
- 在浏览器按 F12 → Console 查看错误信息

## 📊 功能说明

| 页面 | 功能 |
|------|------|
| **总览** | 核心指标卡片 + 用户趋势、流量来源、页面排行图表 |
| **流量分析** | 每日趋势、来源/媒介分布、设备类型、明细表格 |
| **搜索数据** | GSC 点击/展示/CTR/排名、热门查询和页面 |
| **转化漏斗** | 事件趋势分析、事件明细列表 |
| **数据导入** | 支持 CSV 上传（Clarity 录制数据、CRM 填单数据） |
| **策略建议** | 基于数据的自动分析建议，支持筛选和导出报告 |

---

## 📤 导入 Clarity / CRM 数据

1. 切换到「数据导入」页面
2. 拖拽 CSV 文件到上传区域，或点击选择文件
3. 数据自动存入浏览器本地，后续可在分析中参考

---

## 📄 导出报告

在「策略建议」页面点击「导出报告」，生成包含核心指标 + 建议的文本文件。
